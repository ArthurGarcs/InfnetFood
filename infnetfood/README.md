# InfnetFood

Projeto final da disciplina **Desenvolvimento Mobile com React Native**.

O aplicativo **InfnetFood** foi desenvolvido em **React Native**, com foco em simular um app de pedidos e delivery.

## Funcionalidades implementadas

- Tela de login com validação básica
- Navegação entre telas com React Navigation
- Listagem de categorias com FlatList
- Listagem de produtos por categoria
- Tela de detalhes do produto
- Carrinho de compras com cálculo de total
- Tela de perfil com dados mockados
- Tela de pedidos com FlatList
- Mapa simulado com restaurantes no Centro do Rio de Janeiro
- Tela de detalhes do restaurante
- Tela de checkout
- Fluxo de autenticação simples
- Feedback visual ao adicionar item e confirmar pedido
- Tela de configurações com tema claro e escuro
- Notificações locais simuladas
- Integração com API externa de CEP (ViaCEP)

## Tecnologias utilizadas

- React Native
- Expo
- React Navigation
- Expo Notifications
- API ViaCEP
